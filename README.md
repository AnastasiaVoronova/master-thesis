# eNPS Classifier

Система автоматической классификации открытых комментариев eNPS-опросов. Проект объединяет три компонента: encoder-модели (BERT-based двухступенчатый пайплайн), decoder-модели (LLM-классификация) и веб-приложение.

---

## Структура репозитория

```
master-thesis/
├── encoder-models/       # BERT-based двухступенчатый классификатор
├── decoder-models/       # LLM-классификация (DeepSeek, Gemini, Qwen, MiniMax)
└── app/                  # Flask веб-приложение для конечных пользователей
```

---

## Encoder Models — BERT-based двухступенчатый классификатор

### Архитектура

Двухступенчатый пайплайн:

- **Stage 1 (binary):** определяет наличие категории неудовлетворённости (`label > 1`) или её отсутствие (`label ≤ 1`)
- **Stage 2 (multiclass):** классифицирует конкретную категорию (только для позитивных предсказаний Stage 1)

Каждая ступень может использовать независимый backbone.

## Decoder Models — LLM-классификатор

### Подход

Zero-shot классификация через API: LLM получает текст комментария и промпт с описанием всех 30 категорий, возвращает одну метку. Оценка ведётся по двум метрикам:

- **Multiclass** — точное совпадение категории (0 и 1 взаимозаменяемы)
- **Binary** — определение наличия/отсутствия неудовлетворённости

## Приложение (eNPS App)

Flask веб-приложение для классификации открытых комментариев eNPS. Использует обученные модели BERT (Stage 1 + Stage 2) в виде финальных чекпоинтов.

**Системные требования:**

| | Mac | Windows |
|-|-----|---------|
| ОС | macOS 10.15+ | Windows 10/11 |
| RAM | 16 ГБ | 16 ГБ |
| Диск | 8 ГБ | 8 ГБ |
| GPU (опционально) | — | NVIDIA 8 ГБ VRAM |

**Установка:**

```bash
# Mac
curl -fsSL https://raw.githubusercontent.com/AnastasiaVoronova/master-thesis/main/app/Install.command | bash

# Windows — запустить Install.bat
```

После установки на рабочем столе появится ярлык **eNPS_App**.

**Использование:**

1. Загрузите `.xlsx` (drag & drop или выбор файла)
2. Сопоставьте колонки Score, A1/C1, A2/C2
3. Настройте порог классификации (0–1, по умолчанию 0.5)
4. Запустите классификацию и скачайте результат

Результирующий файл содержит исходные данные с двумя добавленными колонками категорий.
