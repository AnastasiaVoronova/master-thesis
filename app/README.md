# ENPS Классификатор

Приложение для автоматической классификации открытых комментариев eNPS-опросов.

---

## Требования

**Mac:** macOS 10.15+
**Windows:** Windows 10/11, NVIDIA GPU (опционально, для ускорения)

**Память:**
- RAM: не менее 16 ГБ
- Диск: не менее 8 ГБ свободного места (модели занимают ~5 ГБ)
- VRAM (при использовании GPU): не менее 8 ГБ

---

## Установка

### Mac

Откройте Terminal и выполните:
```bash
curl -fsSL https://raw.githubusercontent.com/AnastasiaVoronova/master-thesis/main/app/Install.command | bash
```

После установки на рабочем столе появится ярлык **eNPS_App**.

### Windows

1. Скачайте файл `Install.bat` из репозитория
2. Запустите двойным кликом

После установки на рабочем столе появится ярлык **eNPS_App**.

> При наличии NVIDIA GPU установка автоматически подберёт версию PyTorch с поддержкой CUDA

---

## Запуск

Двойной клик по ярлыку **eNPS_App** на рабочем столе.

Браузер откроется автоматически после загрузки моделей. При первом запуске это может занять некоторое время.

> Не закрывайте окно терминала во время работы приложения

---

## Использование

1. **Загрузите файл** — перетащите `.xlsx` в зону загрузки или выберите вручную
2. **Сопоставьте колонки** — укажите столбцы с оценкой (Score), первым и вторым открытыми вопросами
3. **Настройте порог** — чувствительность классификации от 0 до 1 (по умолчанию 0.5)
4. **Запустите классификацию** — нажмите кнопку и дождитесь завершения
5. **Скачайте результат** — готовый файл появится в истории запусков

Результирующий `.xlsx` содержит исходные данные с двумя добавленными колонками категорий.

---

## Удаление

### Mac
```bash
bash ~/eNPS_App/Uninstall.command
```

### Windows

Запустите `Uninstall.bat` из папки `%USERPROFILE%\eNPS_App`.
